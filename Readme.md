# 💰 Expense Tracker App

A simple Expense Tracker web app built using **HTML, CSS, JavaScript, and Bootstrap**.
It helps users add, edit, and delete expenses with data stored in **localStorage**.

---

## 🚀 Features

* Add new expense (Amount, Description, Category)
* Edit existing expense
* Delete expense
* Cancel edit option
* Show total expense
* Data saved in browser using localStorage
* Simple and responsive UI with Bootstrap

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Bootstrap 5
* localStorage (for data persistence)

---

## 📂 How to Run

1. Download or copy the project files
2. Create a file named `index.html`
3. Paste the code into it
4. Open the file in your browser

---

## 📸 How It Works

1. Enter:

   * Amount
   * Description
   * Category
2. Click **Add Expense**
3. Your expense will appear in the list
4. Use:

   * **Edit** to update
   * **Delete** to remove

---

## 💾 Data Storage

* All expenses are stored in **localStorage**
* Data remains saved even after refreshing the page

---

## ⚠️ Validation

* All fields are required
* Error message shown if fields are empty
